🧘 Rebase Services Widget

Embeddable Web Component for booking wellness services

📋 Embed Code

<rebase-services src="https://rebase-flow.lovable.app/widget.js" theme="dark" show-booking="true" ></rebase-services>

⚙️ Configuration Options

| Attribute | Default | Description | |---------------|----------------|--------------------------------| | theme | "dark" | "dark" or "light" | | api-url | Production URL | Override API endpoint | | show-booking | "true" | Enable/disable booking flow | | category | null | Filter to specific category |

ℹ️ Note

This is a development preview. In production, the widget will be bundled as a single JavaScript file that can be embedded on any website using the <rebase-services> custom element.
